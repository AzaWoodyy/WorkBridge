# Contributing

Thanks for your interest in contributing to WorkBridge.

## Development setup
1. Use Node 22 (repo is pinned via `.nvmrc`).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm run dev
   ```

## Pull requests
- Keep changes focused and scoped.
- Update or add tests where relevant.
- Ensure the UI remains stable at narrow widths.
- Avoid committing build artifacts.

## Code style
- TypeScript + React with Tailwind.
- Prefer clarity over cleverness.
- Keep UI changes aligned with `UIUX.MD`.

## Issues
Please include:
- steps to reproduce
- expected vs actual behavior
- screenshots or recordings when helpful
